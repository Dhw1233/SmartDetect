# Frontend + Backend for SmartDetect
this is the python version of FrontEnd and BackEnd for the project SmartDetect,which uses Tran-AD
for Anamoly Detection of electronic mechines.

## env setup
You can use the following command to setup the dependecies within easy reach.
```
pip install -r requirements.txt
```

## app start
use:
```
python app.py
```
to quickly start the backend service.
